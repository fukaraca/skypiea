Features

Skypiea AI is built to showcase robust backend development with Go, containerization, cloud deployments, and more. Below are some of the key features that highlight the project's capabilities:

  • 1. Core AI Service (Go-based REST API)

    Implements RESTful endpoints for AI data processing. Harnesses Go’s concurrency features to efficiently handle requests, ensuring high performance and scalability.

  • 2. Microservices Architecture & Communication

    Splits functionality into smaller Go microservices. Uses gRPC(not for now) or REST alongside a message broker for decoupled, resilient design.

  • 3. Containerization & Orchestration

    Docker-based containers deployed to a Kubernetes cluster (self-managed K3s on AWS free tier t2.micro). Helm charts control versioned and repeatable deployments.

  • 4. AWS Infrastructure

    Integrates AWS services (EC2 and RDS) and leverages AWS best practices for secure, scalable cloud deployments.

  • 5. CI/CD Pipelines

    Automates lint, build, test, and deployment processes. Ensures continuous integration and delivery with GitHub Actions.

  • 6. Comprehensive Testing Strategy

    Ensures unit and integration tests.

  • 7. Server-Side Rendered UI (HTMX + Go Templates)

    Presents a user dashboard using Go’s template engine, HTMX for partial updates, and Bootstrap for a responsive design—offering a straightforward yet modern interface.

  • 8. Observability & Logging

    Implements structured logging and exposes metrics via Prometheus(to be delivered).

  • 9. Security & Authentication

    Employs JWT or session-based authentication, role-based access, and secure coding best practices. Integrates TLS/HTTPS, secret management.
